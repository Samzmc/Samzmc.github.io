# Mingchuan Zou Personal Website

This is a static academic personal homepage intended for GitHub Pages.

## Local preview

Open `index.html` directly in a browser, or run a small local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deploy

Push this repository to:

```text
https://github.com/Samzmc/Samzmc.github.io
```

GitHub Pages will publish it at:

```text
https://samzmc.github.io
```
