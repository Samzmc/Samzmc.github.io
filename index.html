Selected Work

Projects and research experience

pico-decoder-tiny and pico-analyze

Learning dynamics

Independently reproduced a small decoder-only language model around 11M parameters on a single RTX 5080, then built analysis tooling to study training dynamics through CKA, effective rank, and layer-wise weight norm growth.

The project is motivated by a simple question: which parts of a compact language model change most during training, and how can those changes be measured beyond loss curves?

CacheBlend adaptation to TensorRT-LLM

LLM inference

Adapted CacheBlend-style KV-cache reuse ideas toward TensorRT-LLM, focusing on CUDA-level inference paths, memory layout, cache movement, and the practical constraints of high-throughput LLM serving.

Flare DeFi Agent

Agents

Built a tool-using DeFi agent at ETH Oxford 2026 with Claude, MCP, Next.js, FastAPI, and Web3.py. The project explored how LLM agents can interact with blockchain tools through structured, inspectable interfaces.
